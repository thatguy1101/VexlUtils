# VexlUtils

A lightweight utility plugin for modern Paper servers, providing useful tools and quality-of-life features for server owners.

## Features

* Lightweight and optimized
* Easy configuration
* Modern Paper API support
* Clean and simple commands
* Designed for performance and reliability

## Requirements

* Paper 1.21+
* Java 21+

## Installation

1. Download the latest release from the Releases page.
2. Place `VexlUtils.jar` into your server's `plugins` folder.
3. Restart your server.
4. Edit the generated configuration files if needed.
